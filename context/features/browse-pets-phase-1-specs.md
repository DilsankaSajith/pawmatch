# Browse Pets UI Phase 1 Spec

## Overview

This is phase 1 of 2 for the browse-pets UI layout. Use the screenshot referenced below for how it should look. Use the data from the mock data file referenced below. Just import it directly for now until we implement a database.

## Design

- Use the @context/screenshots/browse-pets-ui-main.png and @context/screenshots/browse-pets-ui-sidebar.png for the design.
- Use the original fonts and colors from our project even though the screenshot might have different colors and fonts.
- Use @context/screenshots/Screenshot_1.png, @context/screenshots/Screenshot_2.png, and @context/screenshots/Screenshot_3.png only for design reference.

## Requirements for phase 1

- Collapsible sidebar
- Pet cards with image, name, age, gender, breed, location, and a favorite button
- User avatar area at the left bottom
- Drawer icon to open/close sidebar
- Always a drawer on mobile view
